Portfolio

A glimpse of some of my top projects ranging from data analysis with SQL and python, to visualization with Tableau, data mining with SAS, ETL with SSIS, and web development.

Data Cleaning with SQL

This project involves the utilization of SQL statements to clean and transform raw Nashville housing data, ensuring its suitability for analysis.

Prediction of house prices in Arizona

This project entails utilizing advanced data mining techniques, including regression, decision trees, k-nearest neighbor, clustering, gradient boosting, and model ensembling, using SAS Enterpise Miner to develop an algorithm that accurately predicts house prices in the Arizona housing market.

Movie Correlation with Python

This project centers around Python-based data cleaning and exploration, specifically targeting the variables influencing the gross revenue of movies. It entails conducting statistical analysis, with a primary emphasis on correlation, to unveil relationships between the variables and the movie's revenue.

Tableau Dashboards

Tableau dashboards on different projects.

Covid 19 Data Exploration

This project involves leveraging SQL Server to perform exploratory analysis on global COVID-19 data. The aim is to delve into various aspects of the pandemic, such as case counts, deaths, and vaccination rates, to gain insights and trends that contribute to a comprehensive understanding of the global health crisis.

Blockchain and Web 3.0 projects

My works with blockchain and Web 3.0 include a NFT viewer website which is a platform users can use to view the NFT's they or other users holds in their wallet. I also developed using ChatCPT an NFT minting website whereby users can mint their own NFT's with just the click of a button.

UARK Enterprise Systems Data management

Project undertaken for the Enterprise Systems Department of UARK, who faced challenges including the inability to track student accounts, lack of usage analytics, and difficulty in data retrieval. To address these inefficiencies, an IT solution was implemented to redesign the entire process and establish a more efficient system. Tools such as MSSQL, SQL Server Integration Services, and Python scripts were utilized to extract, transform, and load the data, enabling the client to capture the necessary information and overcome the aforementioned challenges.

Dillard's Customer Analysis

In this project, real company data from Dillard's is utilized to support management decision-making. SQL queries are employed to extract and transform data, while Python is utilized for analysis and visualization. The objective is to provide actionable insights to assist Dillard's management in determining the optimal stores, items, and timing for in-store promotions.

Kaggle Web Scraping with Python

This project entails the extraction and storage of user data from Kaggle, a popular platform for data science and machine learning. The primary objective is to collect relevant user information from Kaggle, utilizing web scraping techniques, and store it securely in a database for further analysis. The acquired dataset will serve as a valuable resource for conducting comprehensive data analysis, exploring patterns, trends, and insights within the user data.
